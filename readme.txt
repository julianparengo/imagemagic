imagemagic

- Preparations
    Please install python3.6
        https://repo.anaconda.com/archive/Anaconda3-5.2.0-Windows-x86_64.exe
		
    Install requirements.txt
         pip install -r requirements.txt
	Second, please install Scribus
		https://sourceforge.net/projects/scribus/files/scribus-devel/1.5.1/scribus-1.5.1-windows-x64-r2.exe/download

Now, you can run from Anaconda Console:

python imagemagic.py ./Input ./PNGOutput ./PDFOutput grass700x700.jpg watermark.png 123
